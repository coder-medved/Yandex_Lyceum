    function checkExternalErrorInner(str, errorItems) {

<section class="material__annotation">
<h2>Аннотация</h2>
<p>Во&nbsp;втором уроке мы&nbsp;впервые познакомимся с&nbsp;одной из&nbsp;фундаментально важных тем в&nbsp;программировании&nbsp;— условным оператором. Он&nbsp;позволяет организовать ветвление в&nbsp;вашей программе (выполнение одной ветки кода в&nbsp;зависимости от&nbsp;условия).</p>
</section>
<section class="material__chapter">
<h2 id="1">Повторение</h2>
<p>На&nbsp;прошлом уроке мы&nbsp;познакомились с&nbsp;переменными. Переменная имеет имя и&nbsp;значение. Имя переменной может начинаться только с&nbsp;буквы и&nbsp;включать в&nbsp;себя буквы, цифры и&nbsp;символ подчеркивания. Имя переменной должно отражать ее&nbsp;назначение.</p>
<p>Чтобы задать переменной значение, необходимо после знака равно (оператора присваивания) указать значение переменной.</p>
<p>Еще значение переменной можно получить из&nbsp;ввода. Для этого используем команду <code>input()</code>. В&nbsp;этом случае значение переменной задает пользователь.</p>
</section>
<section class="material__chapter">
<h2 id="2">Условный оператор</h2>
<section class="material__note">
<p>Условный оператор используется, когда некая часть программы должна быть выполнена, только <strong>если верно</strong> какое-либо условие. Для записи условного оператора используются ключевые слова <var>if</var>&nbsp;и&nbsp;<var>else</var> («если», «иначе»), двоеточие и&nbsp; <strong>отступ в&nbsp;четыре пробела</strong>.</p>
</section>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">if</span> условие<span class="token punctuation">:</span>
    Действия<span class="token punctuation">,</span> если условие верно
<span class="token keyword">else</span><span class="token punctuation">:</span>
    Действия<span class="token punctuation">,</span> если условие неверно
</code></pre>
<section class="material__note">
<p class="material__note-heading"><strong>PEP&nbsp;8</strong></p>
<p>Отступ в&nbsp;четыре пробела принят в&nbsp;сообществе Python (PEP&nbsp;8). При этом программа может работать и&nbsp;при других вариантах, но&nbsp;читать ее&nbsp;будет неудобно. Пробелы&nbsp;— самый предпочтительный метод отступов.</p>
<p>Табуляция должна использоваться только для поддержки кода, написанного с&nbsp;отступами с&nbsp;помощью табуляции.</p>
<p>Python 3&nbsp;запрещает смешивание табуляции и&nbsp;пробелов в&nbsp;отступах.</p>
</section>
<p>Рассмотрим пример:</p>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Введите пароль:'</span><span class="token punctuation">)</span>
password <span class="token operator">=</span> <span class="token builtin">input</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">if</span> password <span class="token operator">==</span> <span class="token string">'qwerty'</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Доступ открыт.'</span><span class="token punctuation">)</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Ошибка, доступ закрыт!'</span><span class="token punctuation">)</span>
</code></pre>
<p>Обратите внимание: в&nbsp;начале условного оператора <var>if</var>&nbsp;выполняется сравнение, а&nbsp;не&nbsp;присваивание. Разница вот в&nbsp;чем.</p>
<section class="material__note">
<p class="material__note-heading"><strong>Сравнение</strong></p>
<p>Сравнение&nbsp;— это проверка, которая не&nbsp;меняет значение переменной (в&nbsp;сравнении может вообще не&nbsp;быть переменных), а&nbsp;присваивание&nbsp;— команда, которая меняет значение переменной.</p>
<p>Для сравнения нужно использовать двойной знак равенства: <code>==</code>.</p>
<p>Также заметьте, что после <var>else</var> никогда не&nbsp;пишется никакого условия.</p>
</section>
<p>Другой пример:</p>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Представься, о незнакомец!'</span><span class="token punctuation">)</span>
name <span class="token operator">=</span> <span class="token builtin">input</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">if</span> name <span class="token operator">==</span> <span class="token string">'Цезарь'</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Аве, Цезарь!'</span><span class="token punctuation">)</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Приветик.'</span><span class="token punctuation">)</span>
</code></pre>
<p>В&nbsp;качестве условия можно использовать и&nbsp;другие операции отношения:</p>
<pre class=" language-python"><code class=" language-python"><span class="token operator">&lt;</span> меньше
<span class="token operator">&gt;</span> больше
<span class="token operator">&lt;=</span> меньше или равно
<span class="token operator">&gt;=</span> больше или равно
<span class="token operator">==</span> равно
<span class="token operator">!=</span> не равно
</code></pre>
<section class="material__note">
<p class="material__note-heading"><strong>PEP&nbsp;8</strong></p>
<p>Все операции отношения оформляются с&nbsp;помощью симметричных пробелов.</p>
<p><strong>Правильно:</strong></p>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">if</span> bird <span class="token operator">==</span> <span class="token string">"Тук-тук"</span><span class="token punctuation">:</span>
</code></pre>
<p><strong><span style="text-decoration: line-through;">Неправильно:</span></strong></p>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">if</span> bird<span class="token operator">==</span><span class="token string">"Тук-тук"</span><span class="token punctuation">:</span>
</code></pre>
</section>
<p>Объекты любой однородной группы можно сравнивать между собой. Подумайте над тем, как можно сравнивать, например, строки.</p>
</section>
<section class="material__chapter">
<h2 id="3">Сложное условие. Логические операции</h2>
<p>Иногда в&nbsp;условном операторе нужно задать сложное условие. Для этого можно использовать логические операции <strong>and</strong> («и»), <strong>or</strong>&nbsp;(«или») и&nbsp;<strong>not</strong> («не»).</p>
<section class="material__note">
<p>Чтобы задать одновременное выполнение двух условий, используем <var>and</var> («и»), если достаточно выполнения одного из&nbsp;двух вариантов (или обоих сразу)&nbsp;— используем&nbsp;<var>or</var> («или»), а&nbsp;если нужно убрать какой-то вариант&nbsp;— <var>not</var> («не»).</p>
</section>
<p>Приоритет выполнения операций:</p>
<pre><code>1. not
2. and
3. or
</code></pre>
<p>Если нужно изменить приоритет операций или вы&nbsp;забыли правила, используйте скобки.</p>
<p>Например, вот так можно проверить, что оба условия выполнены:</p>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Как называются первая и последняя буквы греческого алфавита?'</span><span class="token punctuation">)</span>
greek_letter_1 <span class="token operator">=</span> <span class="token builtin">input</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
greek_letter_2 <span class="token operator">=</span> <span class="token builtin">input</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">if</span> greek_letter_1 <span class="token operator">==</span> <span class="token string">'альфа'</span> <span class="token keyword">and</span> greek_letter_2 <span class="token operator">==</span> <span class="token string">'омега'</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Верно.'</span><span class="token punctuation">)</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Неверно.'</span><span class="token punctuation">)</span>
</code></pre>
<p>Ниже еще несколько примеров.</p>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Как греки или римляне называли главу своего пантеона - бога грома?'</span><span class="token punctuation">)</span>
ancient_god <span class="token operator">=</span> <span class="token builtin">input</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">if</span> ancient_god <span class="token operator">==</span> <span class="token string">'Зевс'</span> <span class="token keyword">or</span> ancient_god <span class="token operator">==</span> <span class="token string">'Юпитер'</span><span class="token punctuation">:</span>

    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Неверно.'</span><span class="token punctuation">)</span>
</code></pre>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Введите имена двух братьев из античных мифов и легенд.'</span><span class="token punctuation">)</span>

<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Неверно.'</span><span class="token punctuation">)</span>
</code></pre>
<p>Обратите внимание: если программу из&nbsp;предыдущего примера вставить в&nbsp;IDE Wing, часть кода условного оператора будет выходить за&nbsp;ограничительную красную черту среды.</p>

<section class="material__note">
<p class="material__note-heading"><strong>PEP&nbsp;8</strong></p>
<p>По&nbsp;стандарту PEP&nbsp;8&nbsp;длина строки должна быть ограничена максимум 79&nbsp;символами.</p>
<p>Есть несколько способов переноса длинных строк.</p>
<ul>
<li>Использование подразумеваемых продолжений Python внутри круглых, квадратных и&nbsp;фигурных скобок: длинные строки могут быть разбиты на&nbsp;несколько строк, заключенных в&nbsp;скобки.</li>
<li>Использование символа ’\’ (обратный слеш, или бэкслеш) для обозначения места разрыва строки.</li>
</ul>
<p>Мы&nbsp;будем использовать первый способ.</p>
<p>Если после перенесенной строки идет один или несколько вложенных операторов (например, мы&nbsp;переносим строку с&nbsp;условием в&nbsp;операторе&nbsp;<var>if</var>), отступ у&nbsp;перенесенной части должен быть на четыре пробела больше, чем у&nbsp;вложенного оператора.</p>
<p>Сделайте правильные отступы для перенесенной строки.</p>
</section>
<p>Тогда представленный выше программный код может быть записан так:</p>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Введите имена двух братьев из античных мифов и легенд.'</span><span class="token punctuation">)</span>

        <span class="token keyword">and</span> <span class="token punctuation">(</span>brother2 <span class="token operator">==</span> <span class="token string">'Поллукс'</span> <span class="token keyword">or</span> brother2 <span class="token operator">==</span> <span class="token string">'Полидевк'</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Верно.'</span><span class="token punctuation">)</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Неверно.'</span><span class="token punctuation">)</span>
</code></pre>
<p>Рассмотрим еще несколько примеров.</p>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Введите любые два слова, но это не должны быть "белый" и "медведь" разом.'</span><span class="token punctuation">)</span>

    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Верно.'</span><span class="token punctuation">)</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Неверно.'</span><span class="token punctuation">)</span>
</code></pre>
<p>А&nbsp;теперь попробуйте решить задачи: «Елочка, гори», «Елочка-2», «Елочка-3».</p>
</section>
<section class="material__chapter">
<h2 id="4">Вложенные условия</h2>
<section class="material__note">
<p class="material__note-heading"><strong>Блок кода</strong></p>
<p>В&nbsp;команде <var>if</var>&nbsp;при выполнении условия можно выполнять более одной команды. Для этого все их&nbsp;необходимо выделить отступом. Такая запись называется <strong>блоком кода</strong>. По&nbsp;отступам интерпретатор определяет, при выполнении каких условий какие команды исполнять. Аналогично можно делать и&nbsp;для команды <var>else</var>.</p>
</section>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Представься, о незнакомец!'</span><span class="token punctuation">)</span>
name <span class="token operator">=</span> <span class="token builtin">input</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">if</span> name <span class="token operator">==</span> <span class="token string">'Цезарь'</span> <span class="token keyword">or</span> name <span class="token operator">==</span> <span class="token string">'Caesar'</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Аве, Цезарь!'</span><span class="token punctuation">)</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Слава императору!'</span><span class="token punctuation">)</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Приветик.'</span><span class="token punctuation">)</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Погода сегодня хорошая.'</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Засим - заканчиваем.'</span><span class="token punctuation">)</span>
</code></pre>
<p>Перед последней строчкой нет отступа, это означает, что она будет выполнена в&nbsp;конце работы программы в&nbsp;любом случае. А&nbsp;вот две предыдущие строчки будут выполнены, только если условие <var>if</var>&nbsp;окажется ложным.</p>
<p>Блоки кода в&nbsp;Python очень гибко устроены: внутри них можно писать любой другой код, в&nbsp;том числе условные операторы. Среди команд, которые выполняются, если условие <var>if</var>&nbsp;истинно («внутри&nbsp;if») или ложно («внутри else»), могут быть и&nbsp;другие условные операторы. Тогда команды, которые выполняются внутри этого внутреннего <var>if</var>&nbsp;или <var>else</var>, записываются с&nbsp;дополнительным отступом.</p>
<p>Изучите пример ниже. <var>elif</var>&nbsp;— это короткая запись для «<code>else: if</code>». Если не&nbsp;пользоваться короткой записью, if&nbsp;пришлось&nbsp;бы писать на&nbsp;отдельной строчке и&nbsp;с&nbsp;отступом (а&nbsp;все, что внутри этого <var>if</var>,&nbsp;— с&nbsp;дополнительным отступом). Это не&nbsp;очень удобно, и&nbsp;<var>elif</var> избавляет от&nbsp;такой необходимости.</p>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Представься, о незнакомец!'</span><span class="token punctuation">)</span>
name <span class="token operator">=</span> <span class="token builtin">input</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">if</span> name <span class="token operator">==</span> <span class="token string">'Цезарь'</span> <span class="token keyword">or</span> name <span class="token operator">==</span> <span class="token string">'Caesar'</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Аве, Цезарь!'</span><span class="token punctuation">)</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'В честь какого бога устроим сегодня празднество?'</span><span class="token punctuation">)</span>
    god <span class="token operator">=</span> <span class="token builtin">input</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
    <span class="token keyword">if</span> god <span class="token operator">==</span> <span class="token string">'Юпитер'</span><span class="token punctuation">:</span>
        <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Ура Громовержцу!'</span><span class="token punctuation">)</span>
    <span class="token comment"># если оказалось, что имя бога не 'Юпитер', то проверяем, </span>
    <span class="token comment"># не равно ли оно строке 'Минерва'</span>
    <span class="token keyword">elif</span> god <span class="token operator">==</span> <span class="token string">'Минерва'</span><span class="token punctuation">:</span>
        <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Ура мудрой воительнице!'</span><span class="token punctuation">)</span>
    <span class="token comment"># следующая строка будет выполнена, </span>
    <span class="token comment"># только если имя бога не 'Юпитер' и не 'Минерва'</span>
    <span class="token keyword">else</span><span class="token punctuation">:</span>
        <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Бога по имени'</span><span class="token punctuation">,</span> god<span class="token punctuation">,</span> <span class="token string">'мы не знаем, но слово Цезаря - закон.'</span><span class="token punctuation">)</span>
    <span class="token comment"># эта команда будет выполнена независимо от того, </span>
    <span class="token comment"># какое имя бога ввёл пользователь, если только изначально</span>
    <span class="token comment"># он представился Цезарем</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Слава императору!'</span><span class="token punctuation">)</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Приветик.'</span><span class="token punctuation">)</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Погода сегодня хорошая.'</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Засим - заканчиваем.'</span><span class="token punctuation">)</span>
</code></pre>
<p>А&nbsp;более простой вариант этой программы теперь попробуйте написать самостоятельно.</p>
</section>
<section class="material__chapter">
<h2 id="5">Операции над строками</h2>
<p>Во&nbsp;всех примерах, которые мы&nbsp;рассматривали, переменные хранили строки. Мы&nbsp;вводили, выводили и&nbsp;хранили строки. Кроме уже описанных действий, строки еще можно складывать.</p>
<p>Давайте попробуем:</p>
<pre class=" language-python"><code class=" language-python">x <span class="token operator">=</span> <span class="token string">'10'</span>
y <span class="token operator">=</span> <span class="token string">'20'</span>
z <span class="token operator">=</span> x <span class="token operator">+</span> y
<span class="token keyword">print</span><span class="token punctuation">(</span>z<span class="token punctuation">)</span>
</code></pre>
<section class="material__note">
<p class="material__note-heading"><strong>PEP&nbsp;8</strong></p>
<p>И&nbsp;опять немного рекомендаций по&nbsp;оформлению (PEP&nbsp;8): ставьте пробелы вокруг знаков операций (*, +, - и&nbsp;т.&nbsp;д.)</p>
<p><strong>Правильно:</strong></p>
<pre class=" language-python"><code class=" language-python">z <span class="token operator">=</span> x <span class="token operator">+</span> y
</code></pre>
<p><strong><span style="text-decoration: line-through;">Неправильно:</span></strong></p>
<pre class=" language-python"><code class=" language-python">z <span class="token operator">=</span> x<span class="token operator">+</span>y
</code></pre>
</section>
<p>В&nbsp;данной программе мы&nbsp;задали переменным <var>x</var>&nbsp;и&nbsp;<var>y</var>&nbsp;значение, переменной <var>z</var>&nbsp;присвоили значение результата сложения <var>x</var>&nbsp;и&nbsp;<var>y</var>.</p>
<p>Результатом выполнения программы будет строка <strong>1020</strong>.</p>
<section class="material__note">
<p class="material__note-heading"><strong>Конкатенация</strong></p>
<p>Операция сложения для строк выполняет конкатенацию двух строк, то&nbsp;есть склеивает их&nbsp;содержимое вместе.</p>
<p>Например: операция <code>«При» + «вет»</code> в&nbsp;результате даст строку <code>«Привет»</code>.</p>
</section>
<p>Обратите внимание: запись: <code>x + y = z</code>&nbsp;недопустима. Оператор присваивания ожидает слева переменную, которой надо присвоить значение, а&nbsp;в&nbsp;правой части находится значение или выражение, которое надо сначала вычислить, а&nbsp;затем присвоить.</p>
<p>Мы&nbsp;могли сократить нашу программу и&nbsp;написать в&nbsp;таком виде:</p>
<pre class=" language-python"><code class=" language-python">x <span class="token operator">=</span> <span class="token string">'10'</span>
y <span class="token operator">=</span> <span class="token string">'20'</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>x <span class="token operator">+</span> y<span class="token punctuation">)</span>
</code></pre>
<p>Результат будет такой&nbsp;же, проверьте. Оператор <code>print()</code> сначала вычислил значение выражения <code>x + y</code>, а&nbsp;потом вывел на&nbsp;экран полученное значение.</p>
<p>А&nbsp;еще такой результат можно получить вот таким образом:</p>
<pre class=" language-python"><code class=" language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'10'</span> <span class="token operator">+</span> <span class="token string">'20'</span><span class="token punctuation">)</span>
</code></pre>
<section class="material__note">
<p class="material__note-heading"><strong>Дублирование</strong></p>
<p>Для строк также можно выполнять умножение. Умножать можно строку на&nbsp;число или число на&nbsp;строку. Операция называется дублирование. В&nbsp;результате нее начальная строка будет повторена заданное количество раз.</p>
<p>Например: <code>3 * "20"</code> то&nbsp;же, что и <code>"20" * 3</code>&nbsp;и&nbsp;, результат будет <code>202020</code> и&nbsp;в&nbsp;том, и&nbsp;в&nbsp;другом случае.</p>
</section>
<p>Примеры использования:</p>
<pre class=" language-python"><code class=" language-python">x <span class="token operator">=</span> <span class="token string">'10'</span>
y <span class="token operator">=</span> <span class="token string">'20'</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>x <span class="token operator">*</span> <span class="token number">2</span> <span class="token operator">+</span> y <span class="token operator">*</span> <span class="token number">3</span><span class="token punctuation">)</span>
</code></pre>
<p>Что будет на&nbsp;экране после запуска такой программы?</p>
</section>
<section class="material__chapter">
<h2 id="6">Команда in</h2>
<p>Теперь рассмотрим новую команду для работы со&nbsp;строками&nbsp;— команду&nbsp;<var>in</var>.</p>
<section class="material__note">
<p class="material__note-heading"><strong>Команда in</strong></p>
<p>Команда <var>in</var>&nbsp;позволяет проверить, что одна строка находится внутри другой.</p>
<p>Например: строка «на» находится внутри строки «сложная задача».</p>
<p>В&nbsp;таком случае обычно говорят, что одна строка является <strong>подстрокой</strong> для другой.</p>
</section>
<pre class=" language-python"><code class=" language-python">text <span class="token operator">=</span> <span class="token builtin">input</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">if</span> <span class="token string">'хорош'</span> <span class="token keyword">in</span> text <span class="token keyword">and</span> <span class="token string">'плох'</span> <span class="token keyword">not</span> <span class="token keyword">in</span> text<span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Текст имеет положительную эмоциональную окраску.'</span><span class="token punctuation">)</span>
<span class="token keyword">elif</span> <span class="token string">'плох'</span> <span class="token keyword">in</span> text <span class="token keyword">and</span> <span class="token string">'хорош'</span> <span class="token keyword">not</span> <span class="token keyword">in</span> text<span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Текст имеет отрицательную эмоциональную окраску.'</span><span class="token punctuation">)</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">'Текст имеет нейтральную или смешанную эмоциональную окраску.'</span><span class="token punctuation">)</span>
</code></pre>
